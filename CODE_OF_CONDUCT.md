# Code of Conduct

Agent Task Loop aims to be a practical and respectful open source project.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being precise about technical problems.
- Keep feedback focused on the work.
- Welcome people who are new to the project.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Dismissive or discriminatory language.
- Publishing private information without consent.
- Repeated disruption of project discussions.

## Enforcement

Project maintainers may moderate, edit, or remove comments and contributions that violate this code of conduct.

To report a concern, email `perfectpan325@gmail.com`.
